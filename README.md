# 🎮 AI-Generated Roguelike

Jeu de roguelike révolutionnaire avec intelligence artificielle générative créant des niveaux, ennemis et histoires infiniment uniques.

## ✨ Fonctionnalités principales

- **Contenu procédural infini** : Chaque partie est unique grâce à l'IA générative
- **Ennemis intelligents** : AI qui s'adapte à votre style de jeu
- **Histoire dynamique** : Une narration qui évolue avec vos actions
- **Difficulté adaptative** : Le jeu s'ajuste en temps réel à votre niveau
- **Graphismes immersifs** : Design moderne et atmosphérique
- **Multiplateforme** : Compatible avec PC, mobile et web

## 🛠️ Stack technologique

- **Game Engine** : Custom engine en JavaScript/TypeScript
- **IA Générative** : GPT-4 pour créer du contenu unique
- **Graphismes** : Canvas API + WebGL
- **Audio** : Web Audio API pour des effets sonores immersifs
- **Multiplateforme** : Web, mobile, desktop

## 📦 Installation

```bash
# Clonez le dépôt
git clone https://github.com/crepin7/ai-roguelike.git
cd ai-roguelike

# Installez les dépendances
npm install

# Lancez le jeu
npm start
```
